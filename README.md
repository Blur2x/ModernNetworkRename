🌐 ModernNetworkRename
一个基于 .NET 8 和 WPF-UI 打造的 Windows 11 风格网络配置重命名工具。
支持 中/英 文。

💡 开发初衷
旨在解决 Windows 每次连接新网络时创建的网络名称混乱问题（例如恼人的“网络 1”、“网络 2”等）。通过本工具，你可以随心所欲地熨平注册表中的褶皱，让网络列表回归整洁与个性化。

✨ 核心特性
现代 UI：深度集成 Mica（云母）材质与圆角设计，与 Windows 11 审美原生适配。

安全编辑：直观编辑系统注册表中的网络名称、描述及类别。

智能清理：一键识别并拔除系统中残留的休眠历史记录，自动锁定并保护当前活跃连接。

权限无忧：内置 UAC 提权申请，双击运行即可自动获取管理员权限。

🛠️ 界面功能详解

右上角点击可切换 中文/英文

注册表键 (Registry Key)：展示该网络配置在底层注册表中的唯一标识，供进阶用户参考。

配置名称 (Profile Name)：核心修改项。修改此处即可改变 Windows 任务栏、设置中心显示的网络名称，让它更有规律。

描述 (Description)：网络配置的备注信息。通常可以忽略，修改后基本不影响视觉显示。

类别 (Category) ‼️重要：

！！注意：该项决定了网络的隔离级别！！

0：公用网络 (Public)

1：专用网络 (Private)

2：域网络 (Domain)

建议：非必要请勿修改，或在 Windows 原生设置界面通过图形化方式调整。

🚀 快速开始
前往 Releases 页面下载最新的单文件可执行程序。

程序要求管理员身份运行（双击将自动触发提权弹窗）。

编辑完成后，必须点击 [保存修改] 按钮方可生效。

🧹 关于“清理休眠网络”
点击此按钮将执行全面清理：除了当前正在连接的网络外，其余所有历史连接信息将从注册表中彻底抹除，还你一个绝对纯净的系统。

风险提示：清理后，Windows 设置界面可能会暂时显示“未连接”，但这属于 UI 滞后，不影响任何网络功能。重启或重新插拔网线后即可恢复正常显示。请在明确知晓操作后果的前提下使用。
